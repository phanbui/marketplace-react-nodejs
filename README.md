To run backend:
- add .env file and add JWT_SECRET_KEY, create cloudinary account and add CLOUDINARY_CLOUD_NAME, CLOUDINARY_API_KEY, CLOUDINARY_API_SECRET
- run mongodb server
- npm install
- node app.js 

To run frontend:
- add .env file and add REACT_APP_BACKEND_URL = http://localhost:8080
- npm install
- npm start
